# cuzillion

> 'cuz there are a zillion pages to check

Inspired by the OG, [Steve Souders' cuzillion](https://github.com/stevesouders/Cuzillion).

## Development

In one terminal...

```bash
yarn
yarn build:watch
```

In another terminal...

```bash
yarn start
```

Open [http://localhost:9801/](http://localhost:9801/).
